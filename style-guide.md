# Style guide

## Fonts

Title fonts:
- BahnSchrift SemiBold SemiCondensed (paint.net)
- The Bold Font (postermywall)

Secondary title fonts: 
- Poppins (postermywall)

Paragraph fonts: 
- Federo (postermywall)
- Shantell Sans (postermywall) (handwritten)

Primary color: #090089
Primary gradient: linear-gradient(135deg, #090089 0%, #1a0fb3 100%)
